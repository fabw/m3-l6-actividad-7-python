# M3_L6_Actividad_7_Fundamentos_de_Programacion_en_Python

## Descripción

Este repositorio contiene el desarrollo de la Actividad N°7 del Módulo 3 de Fundamentos de Programación en Python. La actividad consiste en resolver tres ejercicios prácticos orientados al uso de estructuras de control, validaciones, listas y procesamiento de datos en Python. El enunciado del trabajo corresponde a una práctica en parejas.

## Objetivo

Resolver mediante Python los ejercicios solicitados en la actividad, aplicando lógica de programación básica, validación de entradas, uso de listas, condicionales, ciclos y funciones según corresponda al desarrollo del módulo.

## Contenido del proyecto

El proyecto incluye la resolución de los siguientes ejercicios:

### Ejercicio 1
Programa que solicita un número al usuario y:
- si el número es par entre 2 y 100, imprime los números pares siguientes hasta 100
- si el número es impar entre 1 y 99, imprime los números impares siguientes hasta 99
- si el número ingresado es 0, menor que 0 o mayor que 100, muestra un mensaje de error y vuelve a solicitar el número

### Ejercicio 2
Programa que lee por teclado 5 notas entre 0 y 10 y muestra:
- todas las notas ingresadas
- la nota media
- la nota más alta
- la nota más baja

### Ejercicio 3
Programa que solicita un número de mes y muestra:
- el nombre del mes
- la cantidad de días que tiene
- usando listas
- considerando febrero con 28 días

## Estructura de trabajo utilizada

Para mantener orden y dejar evidencia del proceso, el trabajo se organizó mediante:
- repositorio en GitHub

Este proyecto fue desarrollado de forma individual, utilizando dos cuentas de GitHub propias, con autorización del profesor, para simular la dinámica de trabajo en parejas exigida por la actividad. La organización del trabajo se realizó separando tareas por issues, ramas y contribuciones diferenciadas entre ambas cuentas.
